## Description
Read **two integer numbers**.

Calculate **factorial** of each number.

**Divide the first result by the second** and print the **division formatted to the second** decimal point.

### Example
| **Input** | **Output** |
| --- | --- |
| 5 | 60.00 |
| 3 | |
