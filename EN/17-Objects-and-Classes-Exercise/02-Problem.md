## Description
Using the **Person class**, write a program that reads from the console **N lines of personal information** and then **prints all people** whose **age is more than 30 years** in the following **format**:

\{name\} - \{age\}

### Example
| **Input** | **Output** |
| --- | --- |
| 3 | John - 31 | 
| Peter 12 | Steven - 48 |
| John 31 | |
| Steven 48 | |

### Example

| --- | --- |
| 5 | Nick - 33 | 
| Nick 33 | John - 88 |
| John 88 | Leo - 44 |
| George 22 | |

| Peter 11 | |
