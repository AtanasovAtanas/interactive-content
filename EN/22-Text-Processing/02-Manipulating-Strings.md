# Concatenating

- Using `+` or `+=` operators
```java 
String text = "Hello" + ", " + "SoftUni!";  // "Hello, SoftUni!"
```
- Using `concat()` method
```java live
String greet = "Hello, ";

String name = "SoftUni!";

String result = greet.concat(name);

System.out.println(result); // "Hello, SoftUni!"
```
