## Description
Write a program that **receives an array and number of rotations** you have to perform (first element goes at the end).

**Print the resulting** array.

| **Input** | **Output** |
| --- | --- |
| 51 47 32 61 21 | 32 61 21 51 47 |
| 2 | |
