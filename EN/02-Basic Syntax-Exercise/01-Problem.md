## Description

Write a program that determines whether based on the given age a person is: 

**baby**, **child**, **teenager**, **adult**, **elder**. 

The bounders are:
•	0-2 – baby; 
•	3-13 – child; 
•	14-19 – teenager;
•	20-65 – adult;
•	>=66 – elder; 

All the values are inclusive

### Example
| **Input** | **Output** |
| --- | --- |
| 10 | child |
