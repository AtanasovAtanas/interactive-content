# Types of Sets

**HashSet<E>**

- Initialization:
```java
Set<String> hash = new HashSet<String>();
```


**TreeSet<E>**

- Initialization:
```java
Set<String> tree = new TreeSet<>();
```

**LinkedHashSet<E>**

- Initialization:
```java
Set<String> linkedHashSet = new LinkedHashSet<>();
```
