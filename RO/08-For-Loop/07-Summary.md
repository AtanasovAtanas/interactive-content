# Summary

We can repeat a code block using a `for` loop:
```py live
for i in range(1, 11):
    print(i)
```

We can use a different step in the `for` loop:
```py live
for i in range(1, 20, 2):
    print(i)
```

We can iterate over characters
```py live
for ch in range(65, 71):
    print(chr(ch))
```
