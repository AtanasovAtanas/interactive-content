## Description

Write a program that prints the **type** of an animal according to its **name**.

## Input / Output
- **dog -> mammal**
- **crocodile, tortoise, snake -> reptile**
- **others -> unknown**

## Example

| **Input** | **Output** |
| --- | | --- |
| dog | mammal |

| **Input** | **Output** |
| --- | --- |
| snake | reptile |

| **Input** | **Output** |
| --- | --- |
| cat | unknown |
